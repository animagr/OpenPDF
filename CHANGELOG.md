# Changelog

All notable changes to OpenPDF are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-28

### Added
- PDF viewing with high-quality PDFium rendering, page thumbnails, and Ctrl+scroll zoom (0.25x–5x)
- Merge multiple PDFs and split/extract selected pages, drag-and-drop page reordering
- Inline text editing with font matching against the original document
- Text boxes, freehand drawing, and highlight overlays with adjustable color, size, and opacity
- Reusable signatures: draw, save, and click to place anywhere on a page
- Full-text search with result highlighting, drag-select to copy text
- Print with annotations flattened into the output
- Save Flattened PDF: rasterizes every page at 150 DPI into a fully uneditable document
- Password-protected PDF support with password prompt
- Self-installing EXE: Install/Run dialog, per-user install to %LOCALAPPDATA% (no UAC), PDF file handler registration, Start Menu shortcut, clean uninstall
- Dark theme with green accent
